# 🧠 JavaScript Closure Practice Problems

This repository contains beginner-friendly JavaScript problems focused on **closures, functions, and object manipulation**.


## 📚 Problems Included

### 1️⃣ createHelloWorld
Returns a function that always returns `"Hello World"`.

### 2️⃣ Counter Function
Returns a function that increments value each time it's called.

### 3️⃣ Expect Function
Simple testing function using `toBe` and `notToBe`.

### 4️⃣ Advanced Counter Object
Counter with increment, decrement, and reset features.


## 🧠 Concepts Learned
- Closures
- Higher-order functions
- Object methods
- State management in JavaScript
- Error handling
